# Prompts de Personagens - Sprites

## PROTAGONISTA

### Dr. Residente (personagem principal)

```
16-bit pixel art character sprite, young doctor in light blue scrubs, stethoscope around neck, friendly determined expression, standing pose, hospital ID badge, short dark hair, simple clean design, SNES RPG protagonist style, front facing, transparent background
```

**Variacoes de expressao:**
- Feliz: `smiling happy expression`
- Preocupado: `worried concerned expression`
- Concentrado: `focused determined expression`
- Cansado: `tired exhausted expression`
